# Glossary

**RaCX**: Rules-as-Code Exchange Superset.

**Policy lifecycle graph**: A graph linking policy concepts, variables, parameters, decisions, evidence, process steps, tests, traces, and source references.

**Profile**: A subset of RaCX that an engine may conform to.

**Sidecar**: Metadata and conformance files stored alongside an existing engine model without rewriting it.

**Decision service**: A reusable calculation/eligibility unit with declared inputs, outputs, source refs, and trace contract.

**Process binding**: A typed link from a process or case step to a decision service, evidence requirement, notice, review, or action.

**Semantic equivalence**: Agreement on meaning, source refs, rule paths, parameter versions, and traces, not merely equal numerical outputs.

**Runtime AI**: LLM execution during live eligibility, tax, payment, routing, or refusal decisions. Out of scope and discouraged.
