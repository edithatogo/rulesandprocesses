# Agentic extraction prompt draft

Read the attached legal/policy source. Extract candidate RaCX concepts, variables, temporal parameters, decisions, evidence requirements, process steps, notices, and review rights.

For each candidate object, include:

- proposed stable ID;
- label;
- definition;
- source reference;
- confidence;
- ambiguity notes;
- whether human legal review is required.

Do not create authoritative rules. Produce a draft mapping for human review.
