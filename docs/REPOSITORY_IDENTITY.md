# Repository identity

The canonical repository is `edithatogo/rac-conformance` and the public project name is **RaC Conformance**.

`RaC` means Rules-as-Code. The name deliberately does not revive **RaCX** (“Rules-as-Code Exchange Superset”), which remains historical context for the superseded design notebook. The current project is an evidence-first conformance programme with PIC contracts, harnesses, comparative studies, and rules/process demonstrations.

## Persistent identifiers

Published PIC v0.1 and v0.2 JSON Schema `$id` and internal `$ref` values retain the former `https://github.com/edithatogo/rulesandprocesses/...` base. These are versioned identifiers, not current branding links. GitHub redirects the former repository URL, so resolution is preserved without mutating published contract identity.

New contract versions must use `https://github.com/edithatogo/rac-conformance/...`. Current documentation, citations, submission packets, and operational ledgers must use the canonical repository URL.
