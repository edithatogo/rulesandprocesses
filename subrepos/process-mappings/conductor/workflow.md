# Workflow

While incubated, follow the parent `rac-conformance` `AGENTS.md` and
`conductor/workflow.md`.

1. Work only from an approved parent or future standalone track.
2. Pin and verify primary/official sources before encoding assertions.
3. Add tests before implementing validators or adapters.
4. Keep draft mappings in candidate state until human certification.
5. Run the parent quality gates and record exact external/human blockers.
6. Never create a remote, publish, or promote fixtures without the named human
   gate.
