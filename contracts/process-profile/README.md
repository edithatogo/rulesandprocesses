# PIC Process Profile

`pic-process-profile/0.1.0` is the small platform-neutral contract for
source-backed process traces. Its normative schema and semantics remain in
`rac-conformance`. Domain mappings and platform projections consume it from
their own homes and must report representational loss.

- [Schema](0.1.0/schema.json)
- [Specification](0.1.0/SPEC.md)
- [Authority model](AUTHORITY_MODEL.md)
- [Consumer inventory](CONSUMER_INVENTORY.md)
- [Changelog](CHANGELOG.md)

The contract does not replace BPMN, define an ontology, execute legal or
clinical rules, or make an external engine authoritative.
