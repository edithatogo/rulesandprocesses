# Process Profile Changelog

## 0.1.0

- Initial versioned process-profile contract for states, events, transitions,
  actors, human tasks, rule invocations, source assertions, and trace links.
- Added deterministic schema and semantic reference validation.
- Added explicit distinction between observed, derived, proposed,
  human-certified, and executed lifecycle evidence.
- No migration from an earlier process-profile version exists. Future breaking
  changes require a new major version and before/after examples; additive
  changes require compatibility tests and migration notes.
